# Zoopedia App

## Dev

### 1. Clonar el proyecto
```bash
$ git clone <repo-name>
```
### 2. Instalar Yarn
```bash
$ npm install yarn
```
### 3. Instalar dependencias
```bash
$ yarn
```
### 4. Levantar el Backend
```bash
$ yarn backend
```
### 5. Levantar la app
```bash
$ ng s -o
#o
$ yarn start
```
